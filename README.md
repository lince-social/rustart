# Rustart

This is a template to start projects in rust: Ru[st(art)].

---

Check the [CONTRIBUTING.md](CONTRIBUTING.md) and [LICENSE](LICENSE) to understand more about getting envolved.

This project uses [Mise](https://mise.jdx.dev/) to run the commands, like `mise dev`. Mise installs packages, uses them to run the commands and more. Install with:

```bash
curl https://mise.run | sh
```
